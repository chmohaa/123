# Telegram Media Bot (Bot API) — made with ❤️ love ❤️ for feymova.com / t.me/feymovaq

Бот для Telegram, который скачивает медиа по ссылкам (YouTube, Instagram, TikTok и др.) через `yt-dlp` + `gallery-dl` и отправляет файл в Telegram через **Telegram Bot API**. Для сложных источников (например TikTok) можно подключить cookies для `yt-dlp`.

> Сделано с ❤️ любовью ❤️ специально для **feymova.com / t.me/feymovaq**.

---

## Что умеет бот

- Принимает ссылки на видео/медиа и скачивает контент:
  - основным движком: `yt-dlp`
  - резервным движком: `gallery-dl`
- Отправляет файлы через Telegram Bot API, но в боте установлен лимит 300MB на один файл для стабильной скорости и нагрузки.
- Поддерживает форматы вывода:
  - `auto` (как есть, видео отправляется именно как Telegram video)
  - `mp4`
  - `mkv`
  - `mp3`
- Проверяет обязательную подписку на канал перед выдачей медиа.

- Поддерживаемые платформы (ограниченный whitelist):
  - YouTube Shorts
  - TikTok
  - Instagram Reels
  - Pinterest (фото/видео)
  - Spotify (аудио)
  - VK Music, VK Clips
  - Яндекс Музыка
  - Likee
- Бот обрабатывает несколько запросов параллельно (очередь + concurrency).
- Поддерживает optional cookies-файл для `yt-dlp` (`YTDLP_COOKIES_FILE`) — помогает со сложными источниками (TikTok/Shorts и т.д.).
- Поддерживает **русский и английский** языки.
- При первом `/start` просит выбрать язык через inline-кнопки.
- Все пользовательские действия (кроме `/start`) выполняются через нижнее кнопочное меню.
- Есть кнопка смены языка в меню.
- Владелец может:
  - менять общий текст под каждым отправляемым видео/файлом;
  - запускать рассылку всем пользователям.
- Работает в группах/чатах:
  - пользователь вызывает бота через `@username_бота <ссылка>`;
  - бот отправляет медиа сразу в этот же чат.
- Не хранит файлы постоянно: использует временную папку на время загрузки/отправки и удаляет её автоматически.

---

## Как работает сценарий для пользователя

### В личке
1. Пользователь отправляет `/start`.
2. Бот запрашивает язык (только при первом запуске).
3. Бот показывает кнопочное меню.
4. Пользователь нажимает «Скачать видео/медиа» и отправляет ссылку.
5. Бот скачивает, при необходимости конвертирует и отправляет файл. Ссылку можно отправлять сразу, даже без нажатия кнопки скачивания.

### В группе
1. Пользователь пишет: `@username_бота https://...`.
2. Бот проверяет подписку пользователя на обязательный канал.
3. Бот отправляет медиа прямо в группу.

## Техническая архитектура

- Ядро бота: `python-telegram-bot` (Telegram Bot API).
- Хранилище: `SQLite` (`bot.db`):
  - пользователи,
  - язык,
  - формат,
  - временные состояния интерфейса,
  - глобальные настройки (caption под медиа).
- Загрузчики:
  - `yt-dlp` (основной)
  - `gallery-dl` (fallback)
- Конвертация: `ffmpeg`
- Временное хранение: `tempfile.TemporaryDirectory` (автоочистка)

## Требования

- Python 3.10+
- Системные инструменты в `PATH`:
  - `ffmpeg`
  - `yt-dlp`
  - `gallery-dl`

Python-зависимости устанавливаются из `requirements.txt` (включая актуальный `yt-dlp` из pip).

Лимит файла в самом боте: **300MB**.

## Установка на сервере

```bash
cd /path/to/project
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Убедитесь, что внешние утилиты доступны:

```bash
ffmpeg -version
yt-dlp --version
gallery-dl --version
```

---

## Настройка `.env`

Скопируйте шаблон:

```bash
cp .env.example .env
```

Заполните переменные:

- `BOT_TOKEN` — токен от BotFather
- `OWNER_ID` — Telegram user ID владельца
- `REQUIRED_CHANNEL` — канал для обязательной подписки (`@channel` или ID)
- Лимит размера файла фиксирован в коде: 300MB
- `YTDLP_COOKIES_FILE` — необязательный путь к cookies.txt для сложных платформ
- `CONCURRENT_JOBS` — число одновременных загрузок (по умолчанию 8)

## Настройка в BotFather (для групп)

Чтобы бот корректно работал в группах по вызову `@username <ссылка>`:

1. Добавьте бота в группу.
2. В BotFather откройте настройки бота и проверьте режим privacy.
   - Если бот не видит нужные сообщения, отключите Privacy Mode.
3. Дайте боту права на отправку медиа.

### Если TikTok/Shorts не скачивается

1. Экспортируйте cookies в `cookies.txt` (из браузера).
2. Укажите путь в `.env`:

```env
YTDLP_COOKIES_FILE=/opt/tg-media-bot/cookies.txt
```

3. Перезапустите бота.

Это часто решает ошибки доступа/ограничений у TikTok и части YouTube Shorts.
